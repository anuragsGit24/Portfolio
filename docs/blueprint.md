# **App Name**: Portfolio Prodigy

## Core Features:

- Interactive Hero Section: Implement an interactive 3D/WebGL element that reacts to mouse movement.
- Smooth Transitions: Create animated transitions between page sections.
- Project Showcase: Display high-quality project cards with links to detailed case study pages.
- Direct Route Integrations: Implement direct PDF download for resume and links to LinkedIn and GitHub.
- Theme Customization: Offer Light/Dark mode switch with a slick animation.
- Content Personalization: Utilize AI tool to adapt portfolio presentation based on user interaction and engagement metrics, enhancing personalization and relevance dynamically. This tool can determine which projects or skills to highlight, ensuring the most impactful introduction.
- GitHub Integration: Dynamically fetch and display the user's contribution graph or pinned repositories.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey creativity and depth.
- Background color: Very light grey (#F0F0F0) for a clean, modern aesthetic.
- Accent color: Soft lavender (#E6E6FA) to complement the primary color, suggesting innovation.
- Headline font: 'Space Grotesk' sans-serif for headlines, with a modern, techy feel; body font: 'Inter' sans-serif for readability
- Use minimalist line icons for skills, toolkit, and external links.
- Implement a clean, modern layout with generous spacing and clear visual hierarchy.
- Use subtle animations for hover effects, page transitions, and scroll-based interactions.
